About This Project

This is a practice e-commerce site created to demonstrate layout, accessibility, and basic front-end behaviors for a class project.

Implementation Notes

  1. Semantic HTML (header, main, footer, nav).
  2. Accessible search field and skip link.
  3. Cart simulated using localStorage and client-side JS.
  4. Responsive and mobile-friendly layout.
